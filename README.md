# Wait Wan6 Down for OpenWRT

![](https://img.shields.io/badge/license-MIT-brightgreen.svg?style=plastic "License")

# 功能

* 缓解wan6关闭晚于重新拨号导致的[问题](https://github.com/hanwckf/immortalwrt-mt798x/issues/57)
* 图形界面控制
* 支持重试次数与重试间隔配置

# 使用

**仅支持Luci2**

下载ipk，通过opkg安装即可使用
